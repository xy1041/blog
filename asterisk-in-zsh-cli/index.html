星号符号（*）在CLI开发中造成的困难

发表于 2022-11-15 更新于 2025-11-28 分类于 造轮子

背景

在为公司的翻译工具创建CLI时，传入的参数中有星号符号（*）时遇到了无法识别，打印zsh: no matches found的问题

名词解释

* ：星号符号，英文为asterisk，以下简称星号

问题描述

在开发CLI时，将带有星号的字符串作为option的参数传入，发现报错zsh: no matches found，如果没有星号即不会报错
