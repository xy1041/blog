React移动端项目中判断设备的方向

发表于 2022-04-19 更新于 2022-06-21 分类于 React

背景

直播产品的直播页面分为两种布局，当手机设备横过来的时候需要切换为一种类似全屏的布局，所以需要在产品的各个组件中判断是否是横屏状态

思路

尝试使用window.screen相关API来监听设备屏幕的朝向，并使用redux和自定义hooks来全局应用朝向的样式和逻辑

实现
